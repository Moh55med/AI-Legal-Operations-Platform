# AI Legal Operations Platform

A platform for legal firms to manage cases, track deadlines, organize documents, and query legal data using AI.

## Overview

- Case management (create/update/close cases)
- Deadline tracking with alert scheduling
- Document ingestion and search by metadata
- AI querying over case and document data

## Tech Stack

- Python 3.11+
- FastAPI
- `CodeRabbit` CLI (code review/quality checks)
- `spec-kit` (spec → plan → tasks → implementation)
